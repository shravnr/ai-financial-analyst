AI Financial Analyst Chatbot — System Flow

1 · Input & Routing
User types question
CLI prompt via Rich console
Router classifies intent
gpt-5.4-nano → JSON: {type, ticker, company}
Decision
Intent type?
Farewell
Goodbye, exit
Chat
Respond, loop back
Financial ↓
Extract ticker, continue
2 · Company Detection
Decision
Ticker found?
Yes → continue  |  No + company name → "I don't have data for X"  |  No + previous session → reuse  |  No context → "Which company?"
3 · Company Loading
Decision
Ticker indexed in ChromaDB?
Yes ↓
Skip to question validation
No → Ingest & Index
EDGAR download → chunk (3000 chars, section detection) → embed (text-embedding-3-small) → ChromaDB
4 · Question Validation
Decision
Real question or bare ticker?
Bare ticker → "What would you like to know?"  |  Real question → continue
5 · Agentic Tool Loop
LLM decides what to fetch
gpt-5.4-mini reads question + system prompt + 4 tool definitions  |  Up to 3 iterations
Decision
Call tools or generate answer?
If answer ready → skip to guardrails  |  If needs data → call tool(s) below
search_sec_filings
Hybrid: ~20 vector + ~20 BM25 → RRF merge → 8-12 chunks + data boundary
get_financial_data
Live FMP API — LLM picks from 7 categories, last 2 fiscal years each
get_news
Live NewsAPI — 15 articles → keyword relevance → top 5
get_stock_quote
Live FMP — price, volume, market cap
↑ Tool results added to conversation — LLM can call more tools or answer
6 · Guardrails
Rule-based checks
Citation check (uncited numbers?)  |  Number verification (traceable to sources within 1%?)  |  Sufficiency check (thin context?)
LLM grounding check
gpt-5.4-nano extracts top 3 claims, verifies each against source context
Decision
Warnings found?
No ↓
Clean — proceed
Yes → Self-correction
Feed warnings back to LLM → get corrected answer → re-validate → accept only if warnings decrease
7 · Citation Postprocessing & Output
Strip LLM-generated footnotes
Remove any citations/references sections the LLM added on its own
Rebuild citations from real source metadata
SEC → EDGAR URLs  |  FMP → statement type + date  |  News → article links  |  Renumber sequentially
Display answer with citations
Rich Markdown body + citation block → loop back to user input
